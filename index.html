Architecture & Privacy

This protocol extends Zcash‑grade cryptography into a modular third‑layer system, enabling confidential applications while inheriting base‑layer security.

Protocol Features

Designed as a privacy‑first execution layer, the protocol enforces confidentiality by default. Zero‑knowledge proofs validate correctness while keeping state, identities, and balances fully shielded.

Privacy Protocol

The Privacy Protocol governs identity abstraction, state commitments, and proof verification. All interactions are unlinkable unless explicitly disclosed.

Address Model

Shielded and transparent address compatibility allows seamless interoperability while maintaining strict privacy boundaries.

Transactions

Transactions execute privately with only cryptographic commitments finalized on‑chain.

Selective Disclosure

View‑key mechanisms allow users to grant audit access without surrendering control.

Economics

Fixed supply economics combined with deterministic low‑fee execution.

Scalability

Recursive zk‑proof aggregation enables horizontal scalability without compromising security.
